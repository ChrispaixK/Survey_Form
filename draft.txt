labels that describe the purpose of each field with the following ids: id="name-label", id="email-label",

User Story #13: Inside the form element, I can select a field from one or more groups of radio buttons.

User Story #14: Inside the form element, I can select several fields from a series of checkboxes, each of 
which must have a value attribute.

User Story #15: Inside the form element, I am presented with a textarea at the end for additional comments.

User Story #16: Inside the form element, I am presented with a button with id="submit" to submit all my inputs.
